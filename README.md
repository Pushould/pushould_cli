# [Pushould](https://yhoshino11.github.io) CLI and Ruby Gem

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pushould'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pushould
